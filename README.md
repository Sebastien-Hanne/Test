# 🐞 TP de Correction PHP - media-bug

Bienvenue dans ce TP éducatif ! Ce mini-projet PHP contient **des erreurs volontaires** dans l’architecture MVC.

## 🎯 Objectifs pédagogiques

- Identifier les mauvaises pratiques MVC
- Corriger le découplage Model / Controller / View
- Appliquer les bonnes pratiques du cahier des charges (validation, séparation, réutilisabilité)

## 🧩 À corriger

- Requêtes SQL dans les vues
- Traitement de formulaire dans les vues
- Logique métier mal placée dans les modèles
- Contrôleurs trop gros ou mal séparés
- Vues dépendantes des modèles

## ▶️ Lancement

1. Crée une base `media_bug` dans ton MySQL local
2. Exécute le script `install.sql`
3. Lance un serveur PHP :

```bash
php -S localhost:8000
```

4. Accède à [http://localhost:8000](http://localhost:8000)
